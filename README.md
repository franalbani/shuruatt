# shuruatt
Exploring chess forest

## Installation

* `git clone https://github.com/franalbani/shuruatt.git`
* `cd shuruatt`
* `python -m venv venv`
* `. venv/bin/activate`
* `pip install -r requirements.txt`
* Test it:
* `python -c 'from matecitos import matecitos; print(len(matecitos()))'`
* When done:
* `deactivate`
